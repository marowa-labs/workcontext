# Keyboard Shortcuts

This document provides a comprehensive list of keyboard shortcuts available in the ScholarForge AIeditor to help you work more efficiently.

## Text Formatting

| Shortcut                                   | Action                          |
| ------------------------------------------ | ------------------------------- |
| Cmd+B (Mac) / Ctrl+B (Windows)             | Toggle bold formatting          |
| Cmd+I (Mac) / Ctrl+I (Windows)             | Toggle italic formatting        |
| Cmd+U (Mac) / Ctrl+U (Windows)             | Toggle underline formatting     |
| Cmd+Shift+X (Mac) / Ctrl+Shift+X (Windows) | Toggle strikethrough formatting |
| Cmd+Shift+2 (Mac) / Ctrl+Shift+2 (Windows) | Toggle superscript formatting   |
| Cmd+Shift+S (Mac) / Ctrl+Shift+S (Windows) | Toggle subscript formatting     |

## Headings

| Shortcut                                   | Action            |
| ------------------------------------------ | ----------------- |
| Cmd+1 (Mac) / Ctrl+1 (Windows)             | Apply Heading 1   |
| Cmd+2 (Mac) / Ctrl+2 (Windows)             | Apply Heading 2   |
| Cmd+3 (Mac) / Ctrl+3 (Windows)             | Apply Heading 3   |
| Cmd+Shift+Q (Mac) / Ctrl+Shift+Q (Windows) | Toggle blockquote |

## Lists

| Shortcut                                   | Action                       |
| ------------------------------------------ | ---------------------------- |
| Cmd+Shift+2 (Mac) / Ctrl+Shift+2 (Windows) | Create two-column layout     |
| Cmd+Shift+3 (Mac) / Ctrl+Shift+3 (Windows) | Create three-column layout   |
| Cmd+Shift+L (Mac) / Ctrl+Shift+L (Windows) | Toggle bullet list           |
| Cmd+Shift+O (Mac) / Ctrl+Shift+O (Windows) | Toggle ordered/numbered list |
| Cmd+Shift+C (Mac) / Ctrl+Shift+C (Windows) | Toggle checklist             |

## Text Alignment

| Shortcut                                   | Action            |
| ------------------------------------------ | ----------------- |
| Cmd+Shift+L (Mac) / Ctrl+Shift+L (Windows) | Align text left   |
| Cmd+Shift+E (Mac) / Ctrl+Shift+E (Windows) | Align text center |
| Cmd+Shift+R (Mac) / Ctrl+Shift+R (Windows) | Align text right  |
| Cmd+Shift+J (Mac) / Ctrl+Shift+J (Windows) | Justify text      |

## Indentation

| Shortcut                                   | Action          |
| ------------------------------------------ | --------------- |
| Cmd+Shift+- (Mac) / Ctrl+Shift+- (Windows) | Decrease indent |
| Cmd+Shift++ (Mac) / Ctrl+Shift++ (Windows) | Increase indent |

## Insert Elements

| Shortcut                                   | Action                 |
| ------------------------------------------ | ---------------------- |
| Cmd+K (Mac) / Ctrl+K (Windows)             | Insert link            |
| Cmd+Shift+I (Mac) / Ctrl+Shift+I (Windows) | Insert image           |
| Cmd+Shift+G (Mac) / Ctrl+Shift+G (Windows) | Search online images   |
| Cmd+Shift+T (Mac) / Ctrl+Shift+T (Windows) | Insert table           |
| Cmd+Shift+C (Mac) / Ctrl+Shift+C (Windows) | Insert code block      |
| Cmd+Shift+H (Mac) / Ctrl+Shift+H (Windows) | Insert horizontal rule |

## Academic Features

| Shortcut                                   | Action                |
| ------------------------------------------ | --------------------- |
| Cmd+Shift+A (Mac) / Ctrl+Shift+A (Windows) | Add citation          |
| Cmd+Shift+E (Mac) / Ctrl+Shift+E (Windows) | Edit citation         |
| Cmd+Shift+R (Mac) / Ctrl+Shift+R (Windows) | Insert reference list |
| Cmd+Shift+F (Mac) / Ctrl+Shift+F (Windows) | Add footnote          |
| Cmd+Shift+I (Mac) / Ctrl+Shift+I (Windows) | Import document       |

## Note

- On Windows/Linux, replace "Cmd" with "Ctrl"
- Some shortcuts may vary depending on your keyboard layout
- All shortcuts work in the main editor area when it's focused
